## 📌 Overview  
Predicts whether a passenger survived the Titanic disaster using machine learning based on features like age, gender, class, and fare.

---

## 📊 Dataset  
Includes:  
- Pclass, Sex, Age  
- Fare, SibSp, Parch  
- Embarked  
- Survived (0 = No, 1 = Yes)  

---

## 🧹 Preprocessing  
- Handled missing values  
- Encoded categorical data  
- Removed unnecessary columns  
- Train-test split (80/20)  

---

## 🤖 Model  
- Logistic Regression  
- Built using Scikit-learn  

---

## 📈 Results  
- Evaluated using Accuracy, Precision, Recall  
- Key insights:  
  - Females had higher survival rates  
  - Higher class → better survival  
  - Age & fare impacted survival
